# Tic Tac Toe Web Application

## Overview
This project is a simple and interactive **Tic Tac Toe game** developed using **HTML, CSS, and JavaScript**. The application allows two players to play the classic Tic Tac Toe game directly in a web browser. It demonstrates fundamental concepts of front-end web development such as DOM manipulation, event handling, and responsive design.

## Features

- Two-player gameplay
- Interactive and responsive game board
- Automatic winner detection
- Draw condition detection
- Clean and user-friendly interface
- Instant game reset functionality

## Technologies Used

The following technologies were used to develop this project:

- **HTML5** – Structure of the web application  
- **CSS3** – Styling and layout design  
- **JavaScript** – Game logic and interactivity

## Installation and Usage

Follow these steps to run the project locally:

1. Clone the repository
   git clone https://ankit-chauhan-dev.github.io/Tic-Tac-Toe/

2. Navigate to the project directory.

3. Open the `index.html` file in any modern web browser.

4. Start playing the game.
